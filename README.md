# ZionBrowser — Ultra-Lightweight Mobile Browser

**Pure Python stdlib. ZERO external dependencies. ~5MB RAM.**

## Features

- Fetch & display any web page
- Extract links from pages
- Detect HTML forms
- DuckDuckGo search integration
- Cookie management
- Session persistence
- HTTP retry with backoff
- JS-only page detection
- CSRF token extraction

## Build

### Local (requires Android SDK)
```bash
pip install buildozer kivy cython
buildozer android debug
```

### Cloud (GitHub Actions)
Push to `main` branch — APK built automatically.
